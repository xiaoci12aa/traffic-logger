# Comptador de dades de trànsit

[View full project on microbit\.org](https://microbit.org/ca/projects/make-it-code-it/traffic-survey-data-logger)

To code this project yourself, right click on the background of the workspace and choose ‘delete all blocks’, then follow the coding video below. You can find the `forever` and `on start` blocks in the `Basic` section.

### En què consisteix?

Fes servir la teva micro:bit com a comptador de fets que passen al teu voltant, com coses del trànsit, la natura o respostes a preguntes. 

#### Introducció

https://www.youtube.com/watch?v=7Jb-31_72nc

#### Guia de programació

https://www.youtube.com/watch?v=2E2i8WjuX-0

Quan polses un botó, la micro:bit registra el que has vist i emmagatzema les dades en una taula perquè puguis analitzar-les més tard.

Les dades romandran a la teva micro:bit encara que en desconnectis les bateries o el cable USB, de manera que pots estudiar-les quan tornis a l'ordinador.
