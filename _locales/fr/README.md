# Enregistreur de données d'enquête sur le trafic

[Voir le projet complet sur microbit\.org](https://microbit.org/fr/projects/make-it-code-it/traffic-survey-data-logger)

Pour coder ce projet vous-même, cliquez avec le bouton droit de la souris sur l'arrière-plan de l'espace de travail et sélectionnez « Supprimer tous les blocs », puis suivez la vidéo de codage ci-dessous. Vous trouverez les blocs `forever (répéter indéfiniment)` et `on start (au démarrage)` dans la section `Basique`.

### Qu'est-ce que c'est ?

Utilisez votre BBC micro:bit comme enregistreur de données pour interroger des choses que vous voyez, comme différents types de trafic ou de faune, ou des réponses aux questions. 

#### Introduction

https://www.youtube.com/watch?v=7Jb-31_72nc

#### Guide de codage

https://www.youtube.com/watch?v=2E2i8WjuX-0

En appuyant sur un bouton, le micro:bit enregistre ce que vous avez vu et stocke les données dans un tableau pour que vous puissiez les analyser plus tard.

Les données restent sur votre micro:bit même si vous déconnectez les piles ou le câble USB, afin que vous puissiez l'étudier lorsque vous serez de retour sur votre ordinateur.
