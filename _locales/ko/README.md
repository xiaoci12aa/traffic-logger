# 교통 조사 데이터 로거

[View full project on microbit\.org](https://microbit.org/ko/projects/make-it-code-it/traffic-survey-data-logger)

To code this project yourself, right click on the background of the workspace and choose ‘delete all blocks’, then follow the coding video below. You can find the `forever` and `on start` blocks in the `Basic` section.

### 프로젝트 소개

BBC micro:bit 를 데이터 로거로 사용하여 다양한 교통 상황, 야생 동물, 질문에 대한 답변 등 눈에 보이는 모든 것을 조사해 보세요. 

#### 소개

https://www.youtube.com/watch?v=7Jb-31_72nc

#### 프로그래밍 가이드

https://www.youtube.com/watch?v=2E2i8WjuX-0

버튼을 누르면 micro:bit 는 사용자가 본 내용을 기록하고 해당 데이터를 표로 저장하므로 나중에 분석할 수 있습니다.

배터리나 USB 케이블을 분리하더라도 micro:bit 에 데이터가 남아 있으므로 컴퓨터로 돌아와서 공부할 수 있습니다.
